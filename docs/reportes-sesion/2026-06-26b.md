## Decisiones técnicas

| Decisión | Razón |
|---|---|
| `{% block %}` con `{% if %}` dentro | Django compila blocks en tiempo de parse; el `if` fuera del block siempre anulaba el default del padre |
| Child URL = `{{ request.path }}{{ child.slug }}/` | Evita `reverse()` con N slugs de ancestros; limpio y correcto |
| Video primario = primer `video_kind=explicacion` | Un nodo puede tener múltiples videos; el de explicación va arriba |
| `NodeMedia` con delete+recreate en sync | La clave `media:` en YAML indica intención de reemplazar completo, no merge parcial |
| Staff ve nodos no publicados | Permite QA de contenido antes de publicar |
