## Decisiones importantes
- El contenido del YAML (ejemplos, errores_frecuentes) **no genera XP** — es interactivo
  en la UI pero no conecta al sistema de evaluación. Solo el JSONL genera XP/estrellas.
- La `introduccion` es obligatoria en la pauta (campo requerido para recursos nuevos).
